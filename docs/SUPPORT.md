# Support

## Need Help with FinAuditPro?

We are committed to helping auditors make the most out of FinAuditPro. Here are the best ways to get support:

### 1. Documentation
Before asking for help, please review our comprehensive documentation:
- [Installation Guide](docs/INSTALLATION_GUIDE.md)
- [User Manual](docs/USER_MANUAL.md)
- [Troubleshooting Guide](docs/TROUBLESHOOTING.md)

### 2. GitHub Issues (Bugs & Feature Requests)
If you found a bug or have a specific feature request, please open an issue using one of our templates:
- [Report a Bug](https://github.com/aryanyadav/FinAuditPro/issues/new?template=bug_report.md)
- [Request a Feature](https://github.com/aryanyadav/FinAuditPro/issues/new?template=feature_request.md)

### 3. GitHub Discussions (Q&A and General Chat)
For general questions, configuration help, or discussing best practices with AI in auditing, please use GitHub Discussions.
- [FinAuditPro Discussions](https://github.com/aryanyadav/FinAuditPro/discussions)

### 4. Enterprise & Priority Support
If you represent an accounting firm requiring dedicated setup assistance, SLA-backed support, or custom model fine-tuning, please contact our enterprise team directly:
**Email**: enterprise@finauditpro.com
