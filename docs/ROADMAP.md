# Project Roadmap

This roadmap outlines the past, present, and future trajectory of FinAuditPro.

## Phase 1: Core Foundation (v1.0.0) - Current Release
- [x] Complete Offline-First Architecture (SQLite).
- [x] PySide6 Desktop User Interface.
- [x] Basic Local LLM Integration (Ollama).
- [x] PaddleOCR Integration for table extraction.
- [x] ICAI-Standard PDF Audit Report Generation.
- [x] Real-time Executive Dashboards.

## Phase 2: Collaboration & Web Extension (v1.5.0)
- [ ] Migrate database abstraction to support PostgreSQL on LAN.
- [ ] Implement robust Active Directory SSO.
- [ ] Basic internal Web Portal (Read-only dashboard via FastAPI).
- [ ] Add Multi-Language OCR Support.

## Phase 3: Fine-tuned Financial AI (v2.0.0)
- [ ] Implement domain-specific LoRA adapters (fine-tuned on GAAP/IFRS principles).
- [ ] Integration with leading Bank APIs (Plaid, Stripe) for automatic statement reconciliation.
- [ ] Real-time anomaly streaming during client uploads.

## Phase 4: Enterprise Scalability (v3.0.0)
- [ ] Kubernetes-based cloud deployment alternative for non-sensitive data environments.
- [ ] Advanced Graph Database integrations for tracking shell companies and complex corporate structures.
